상단 플로팅 : header, 특정 상단 영역

  • header는 스크롤 시 항상 상단 fixed
  • 특정 상단 fixed 영역은 마크업 시 div.fixed-top 로 감싸고 fixed 시킬 영역이 .fixed-top__inner class를 지정

하단 플로팅 : 플로팅 되지 않고 콘텐츠 하단에 존재하는 버튼

  • jQuery로 콘텐츠 감싸는 영역 높이, 콘텐츠 내용 높이 비교하여 .non-fixed class 추가/제거 하는 방식으로 default position:fixed 상태에서 스크롤 될 경우 position:relative 상태로 변경
  • 최초 진입 시 콘텐츠 내용이 콘텐츠 감싸는 영역 높이 보다 작을 경우 디바이스 하단에 버튼 위치 (하단 position:fixed 상태)
  • 최초 진입 시 콘텐츠 내용이 콘텐츠 감싸는 영역 높이 보다 클 경우 콘텐츠 스크롤 후 하단에 버튼 위치 (position:relative 상태로 변경)
  • 조회,검색 등 무언가의 액션으로 콘텐츠 내용의 높이에 변화가 생길 경우 위의 내용과 같이 버튼 위치 변동

js 요청사항

[최상단에서 화면 스크롤 시 상단 fixed]

  • 1. header 안의 .header__inner 에 is-fixed class 추가 후 header 영역에 style="height:.header__inner 영역 height 값" 추가
  • 페이지 내 .fixed-top 영역이 존재할 경우 해당 .fixed-top 안의 .fixed-top__inner 에 is-fixed class 추가 후 .fixed-top 영역에 style="height:(.fixed-top__inner 영역 height 값)" 추가
  • 화면 최상단으로 이동 시 class 및 height 값 삭제

  • 페이지 내 .floating-btn 영역이 존재할 경우 .floating-btn 영역에 style="height:(.floating-btn__inner 영역 height 값)" 추가
  • 화면 최하단으로 이동하여도 값 유지
