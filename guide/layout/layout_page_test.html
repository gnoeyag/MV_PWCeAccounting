본문 바로가기

  • 상하단 구분되는 타입의 화면일 경우 content-head/content-body 로 구분
  • 상하단 구분없는 화면은 content-head/body 사용하지 않고 각 페이지 디자인에 맞춰 마크업
  • content-body 영역에 default 로 border: 1px solid #D93954 들어가있는 상태
  • content-body 영역에 border 의 color : #dddddd 사용 시 .border-type-2 더블 class 사용
  • content-body 영역에 border 의 color : #D93954, weight : 2px 사용 시 .border-type-3 더블 class 사용
  • content-body 영역에 border 가 필요 없는 경우 .no-line 더블 class 사용
  • 각 div 안에서 각 페이지 디자인에 맞춰 div.content 영역 마크업
