  • js에서 텍스트 전환이 필요한 경우, 다국어 적용을 위해 태그에 data-text 속성 추가 
    <span data-text='{
      "text":"열기|닫기", //적용 텍스트 "|"로 구분
      "show":1 //Default 텍스트 순서로 지정
     }'>open</span>

Accordion

  • Default가 닫힘일 경우 button.accordion__btn에 aria-expanded="false" button.accordion__btn>span close 텍스트 변경, div.accordion__panel에 aria-hidden="true" style="display:none" 적용
  • .accordion에 data-options 추가하여 accordion 응용 
    data-options='{
 "id": "accordion-id", //toggle 버튼과 panel에  id/aria 속성 js에서 생성을 위한 id 추가
 "openType": "single", //single .accordion-item 한개 오픈 / multi .accordion-item 다중 오픈
 "animation": true, //슬라이드 다운 모션
 "animationSpeed":400 //슬라이드 다운 스피드
 }
}'

Select

  • 지원 브라우저 : IE9+ ~ , Chrome, firefox, safari
  • jquery 버전 : 1.7.2 ~ 3.4.1 적용 가능
  • window.ready 이후 .select-wrap > select 가 존재하는 경우 디자인 셀렉트용 div가 생성되며 기본 셀렉트는 display:none로 변경됨
  • 셀렉트에 이벤트를 바인딩 하는 경우는 select , change 두 가지로 바인딩 할 수 있도록 설정되어 있다
  • 셀렉트의 option이 동적으로 생성이 되거나 상태 값이 백단에서 변경이 이루어져야 하는 경우 디폴트 셀렉를 변경 후 jQuery 실렉터및 메소드 호출
    예시 => mvJs.fn.selectmenu.update('#test1'); 셀렉트 커스텀 메소드를 호출하면 디자인 셀렉트도 반영이 된다. header => script 에 간단하게 바인딩된 샘플이 있습니다.
  • select 태그 옵션 설정 
    data-options='{
   "maxHeight": "250", // 셀렉트 메뉴 최대 높이 지정
                       // inherit 설정시 css로 높이값 커스텀 가능
   "direction": "down", // 셀렉트 메뉴 열리는 방향 설정 up/down 디폴트값 down
   "msg": "Default select example" // 셀렉트 메시지 설정
                                   // 옵션이 선택되어 있어야 하는 경우에는 data-msg 옵션 사용하지 않고 select option에 selected 속성 추가
   "multiText": "|" // select > option 텍스트에 구분자를 지정, a > span 태그 생성
                    // "구분 값"은 select > option 텍스트에 사용하지 않는 특수 기호 사용
                    // span 태그에 class 추가 필요 시 select > option 태그에 class="첫번째class 두번째class" 정의, option에 각각 class 적용됨
  }'

Swiper

  • .swiper > .swiper-container > .swiper-wrapper > .swiper-silde 구조를 지켜야 swiper 플러그인 적용됨
  • .swiper-button-next, swiper-button-prev, swiper-pagination, swiper-autoplay 의 위치는 디자인에 따라서 변경해도 무방
  • .swiper-silde에 backface-visibility:hidden;transform:translate3d(0,0,0); 스타일 적용 필요(ie 깜박임 방지)
  • 해당 플러그인은 https://swiperjs.com/swiper-api , https://swiperjs.com/demos 참고
  • .swiper에 data-options 추가하여 swiper 응용 
    data-options='{
 "loop": false, //slide 맨 처음 끝에서도 멈춤없이 이어서 동작 : true/false
 "autoplay": "0", //자동재생 속도 : 0(재생없음) 2000 등
                  // 0 외의 수치값 입력 시 <div class="swiper-autoplay"> 영역 js에서 생성
 "pagination": "fraction", //bullet:인디케이터 형태, fraction: 0/0 페이징처리, all:둘 다
                           //<div class="swiper-pagination"></div> 영역 필요
 "mode": "pc", //PC, MOBILE 모두 swiper 일 때 옵션 작성 안함
               //pc에만 swiper 기능 사용하는 경우 "pc", MOBILE에서만 swiper 기능 사용하는 경우 "mo"
 "effect": "fade", //slide 모션이 fade in out 형태
 "lazy": true, //true lazyload 적용
               //src 속성 적용 이미지 경로, data-src 속성 더미 이미지 경로
 "breakpoints": {
  "1": {
   ex) "slidesPerView" : 2
   //MOBILE에만 적용 할 경우
  },
  "768": {
   ex) "slidesPerView" : 3
   //PC에만 적용 할 경우
  }
 }
 "autoDelete": false, //기본 false : 슬라이드가 1개 있을 경우 스와이프가 생성이 되나 스와이프가 생성이 되지 않는 상태로 대기 개발에서 슬라이드 추가시 자동으로 작동, true : 슬라이드가 1개일시 스와이프가 생성 되지 않음
}'
  • sample3 참고 : 긴 바 형태 추가하려면 <div class="swiper-pagination-progressbar swiper-pagination-horizontal"><span class="swiper-pagination-progressbar-fill"></span></div> 영역 필요
